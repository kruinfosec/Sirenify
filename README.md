# Sirenify
It's an AI-integrated Music Player which can be use to play music create playlist and other stuff like other music play.
What's speacial about it, is that it can analyze the user's mood by using an AI and recommend Music according to that.
It can also filter and organise user's song from his/her created playlist according to the mood and genre of the songs. So that user don't need to search and queue different songs to cater his/her mood.
